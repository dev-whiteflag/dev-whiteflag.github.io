Webapp de Uso Interno

O Webapp foi criado como uma tentativa de melhorar um processo de geração de certificados, os certificados seriam colonizados com informações de um servidor Firebase e transformadas em Excel para Impressão. Foi usado Angular Bleeding edge, junto com bibliotecas do Firebase (includindo Banco de Dados e Autenticação) e outras para geração dos certificados.
